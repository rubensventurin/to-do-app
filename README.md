## Sobre o Projeto

Aplicativo de Lista de Tarefas em React Native criado para fins de aprendizado.

Você pode adicionar tarefas, excluí-las e marcá-las como concluídas.

### Construído com

- Expo
- React Native
- Redux
- Redux Toolkit
- Redux Persist
- Husky
- Prettier
